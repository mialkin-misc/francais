# Ponctuation

> Les règles de ponctuation varient d'une langue à une autre, et même d'un pays
> francophone à un autre.Celles-ci concernent la France et le français.

## Signes

| Signe | Nom                   | Exemple |
| ----- | --------------------- | ------- |
| `,`   | virgule               |         |
| `.`   | point                 |         |
| `...` | points de suspension  |         |
| `:`   | deux-points           |         |
| `;`   | point-virgule         |         |
| `!`   | point d'exclamation   |         |
| `?`   | point d'interrogation |         |
| `[]`  | crochets              |         |
| `()`  | parenthèses           |         |
| `«»`  | guillemets            |         |
| `“”`  | guillemets anglais    |         |
| `—`   | tiret                 |         |
